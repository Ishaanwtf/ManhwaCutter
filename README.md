# Manhwa Cutter

**Manhwa Cutter** is a lightweight tool designed for YouTube manhwa
recap creators and anyone who needs to make multiple precise cuts from a
single image.

## Features

-   **Multiple Cutting Boxes**
    -   Create multiple selection boxes on a single image without
        repeating the process.
    -   Perfect for extracting many panels quickly.
-   **Automatic Box Naming**
    -   Every cut is automatically named in order.
    -   Keeps exported images organized and saves time.
-   **Fast Workflow**
    -   Designed to minimize repetitive work.
    -   Create, adjust, and export cuts efficiently.
-   **High-Resolution Support**
    -   Works well with large and long manhwa images.
-   **Simple Interface**
    -   Easy to learn and use.
    -   Focus on cutting instead of complicated settings.

## Built for YouTube Manhwa Recap Channels

This tool is made with YouTube recap creators in mind. Instead of
cropping one panel at a time, you can prepare multiple cuts in a single
session, making the editing process significantly faster.

Typical workflow:

1.  Load a long manhwa image.
2.  Create multiple cutting boxes.
3.  Export all cuts.
4.  Import them into your video editor.

## Also Useful For

Although designed for manhwa creators, the cutter can also be used for:

-   Manga and comic panel extraction
-   Webtoon editing
-   Creating tutorial images
-   Splitting large screenshots
-   Any project requiring multiple crops from one image
